

# OpenTerminal Agents

OpenTerminal supports different types of agents, each with specific responsibilities and boundaries:

## Operator Agent
The Operator Agent is responsible for executing tasks as requested by the user. It interacts with the CLI and manages the execution environment.

## Guardian Agent
The Guardian Agent enforces security policies and ensures that only authorized operations are performed. It monitors agent activities and can block or approve actions based on predefined rules.

## Developer Agent
The Developer Agent assists in development tasks, such as code generation, debugging, and testing. It provides tools and utilities to help developers work efficiently within the OpenTerminal environment.
